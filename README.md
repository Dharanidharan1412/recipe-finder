# 🍳 Recipe Finder Web Application

## 📌 Project Description

The Recipe Finder is a web-based application that helps users discover recipes based on the ingredients they already have. Users can log in, search for recipes, filter them by cuisine, and view detailed cooking instructions.

This project is built using HTML, CSS, and JavaScript and works completely on the frontend without a backend database.

## ✨ Features

### 🔐 User Login System

Simple login authentication

Guest login option

Demo credentials available

### 🏠 Home Page

Personalized welcome message

Feature overview

Navigation to recipe search

### 🔍 Ingredient-Based Recipe Search

Enter available ingredients

Find recipes that match those ingredients

### 🍽 Recipe Details

Cooking time

Difficulty level

Ingredients list

Step-by-step instructions

### ⭐ Favorites System

Save favorite recipes

Filter recipes to show favorites only

### 🎛 Filters and Sorting

Filter by cuisine (Indian, Italian, Chinese, etc.)

Sort recipes by:

Name

Cooking Time

Difficulty

### 🎨 Modern UI

Responsive layout

Clean design

Animated interface

🛠 Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Application logic and interactivity

Session Storage – Store login session and user data
