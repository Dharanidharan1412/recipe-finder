🍳 Recipe Finder Web Application
📌 Project Description

The Recipe Finder is a web-based application that helps users discover recipes based on the ingredients they already have. Users can log in, search for recipes, filter them by cuisine, and view detailed cooking instructions.

This project is built using HTML, CSS, and JavaScript and works completely on the frontend without a backend database.
